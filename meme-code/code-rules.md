### ### General

Slangython is python-like. The general syntax structure is similar to python. For simplicity, all variables should be in lower case.

### if, elif, else

Python:

```python
if [expression]:
    [statement]
elif:
    [statement]
else:
    [statement]
```

Slangython:

```Slangython
JIC [expression] LOL
    [statement]
JW [expression] LOL
    [statement]
JK LOL
    [statement]
```
